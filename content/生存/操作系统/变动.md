# 你现在应该采用的最终流程

## 1. 重装 Ubuntu 26.04

照之前流程做：

```text
制作启动盘
BIOS 设置
手动分区
安装 Ubuntu Desktop
首次启动更新系统
```

分区仍然建议：

|挂载点|大小|文件系统|
|---|--:|---|
|`/boot/efi`|1GB|FAT32|
|`/`|150GB|ext4|
|swap|64GB|swap|
|`/home`|120GB|ext4|
|`/srv/ai`|剩余全部|ext4|

这个不用变。

## 2. 宿主机基础包仍然要装

安装完系统后：

```bash
sudo apt update
sudo apt full-upgrade -y
sudo reboot
```

然后装基础工具：

```bash
sudo apt install -y \
  build-essential git curl wget unzip jq \
  ca-certificates gnupg lsb-release \
  htop btop nvtop tmux tree ripgrep fd-find \
  python3 python3-venv python3-pip pipx direnv \
  openssh-client rsync
```

这里的 `python3`、`python3-venv`、`python3-pip` 可以保留。

原因：

```text
系统工具可能需要 Python
pipx / direnv / 一些脚本可能需要 Python
mise 自己不等于系统 Python
```

但注意：**不要用宿主机 Python 全局装项目依赖**。

## 5. Docker + NVIDIA Container Toolkit 照旧

Docker 仍然按之前流程安装。

重点仍然是：

```bash
sudo mkdir -p /srv/docker
```

配置：

```bash
sudo tee /etc/docker/daemon.json > /dev/null <<'JSON'
{
  "data-root": "/srv/docker",
  "log-driver": "local",
  "log-opts": {
    "max-size": "20m",
    "max-file": "5"
  },
  "features": {
    "buildkit": true
  }
}
JSON
```

然后：

```bash
sudo systemctl restart docker
sudo usermod -aG docker ai
```

再装 NVIDIA Container Toolkit：

```bash
sudo nvidia-ctk runtime configure --runtime=docker
sudo systemctl restart docker
```

测试：

```bash
docker run --rm --gpus all ubuntu nvidia-smi
```

```bash
mise use --global rust@latest
```

```bash

```

这里的区别是：

```text

```

也就是说，**uv 交给 mise 管**。

# 10. 每个项目加一个 mise.toml

例如：

```bash
cd /srv/ai/projects/torch-test
nano mise.toml
```

内容：

```toml
[tools]
python = "3.12"
uv = "latest"
node = "22"

[env]
HF_HOME = "/srv/ai/cache/huggingface"
TRANSFORMERS_CACHE = "/srv/ai/cache/huggingface"
TORCH_HOME = "/srv/ai/cache/torch"
UV_CACHE_DIR = "/srv/ai/cache/uv"
PIP_CACHE_DIR = "/srv/ai/cache/pip"

[tasks.install]
run = "uv sync"

[tasks.test]
run = "uv run pytest"

[tasks.run]
run = "uv run python main.py"
```

进入项目后：

```bash
mise trust
mise install
mise run install
```

# 11. Ollama / Qdrant / Postgres 仍然用 Docker

```bash
cd /srv/ai/compose
```

```yaml
services:
  ollama:
    image: ollama/ollama:latest
    container_name: ollama

      - "11434:11434"

      - /srv/ai/models/ollama:/root/.ollama

    image: qdrant/qdrant:latest
    container_name: qdrant
    restart: unless-stopped

      - "6333:6333"

      - /srv/ai/datasets/qdrant:/qdrant/storage

    image: pgvector/pgvector:pg17
    container_name: pgvector
    restart: unless-stopped
    environment:
      POSTGRES_USER: ai
      POSTGRES_PASSWORD: change_this_password
      POSTGRES_DB: ai
    ports:
      - "5432:5432"
    volumes:
      - /srv/ai/datasets/postgres:/var/lib/postgresql/data
```

启动：

```bash

```
